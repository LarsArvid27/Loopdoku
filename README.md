# 🎵 Loopdoku - Shakira's Collaboration Puzzle

A logic puzzle game inspired by Sudoku where players help Shakira figure out which song she's writing and where each collaborator is in the studio.

## 🎮 Game Rules

1. **One person per row and column** - Just like Sudoku
2. **Artists only in empty squares** - They can't sit on plants, tables, or shelves
3. **Main artist (Shakira) is alone** - With only one collaborator at a time
4. **Satisfy all clues** - Use the location and song hints to solve the puzzle

## 🎤 Collaborators

- **Maluma** - In a chair (sillón)
- **Beéle** - In the library (biblioteca)
- **Rauw Alejandro** - Next to a plant
- **Beyoncé** - Alone in a corner (alcoba)
- **Carlos Vives** - In the entrance (entrada)

## 🚀 Getting Started

1. Clone the repository
   ```bash
   git clone https://github.com/LarsArvid27/loopdoku.git
   cd loopdoku
   ```

2. Start a local server
   ```bash
   python -m http.server 8000
   ```

3. Open `http://localhost:8000` in your browser
